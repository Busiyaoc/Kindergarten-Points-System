# 班级积分系统（幼儿园版）

这是一个可以托管到 GitHub Pages 的静态网站，不需要后端服务。

## 功能

- 新建班级
- 添加幼儿姓名和头像
- 正反馈行为加分
- 负面行为减分，积分不低于 0
- 幼儿卡片内置精灵养成系统，积分越高形态越成熟、装备越华丽
- 管理员新建行为规则
- 奖励商店新建奖励名称和兑换分值
- 幼儿使用积分兑换奖励
- 兑换记录保存在当前浏览器

## 发布到 GitHub Pages

1. 在 GitHub 新建仓库。
2. 上传 `index.html`、`styles.css`、`app.js`、`.nojekyll` 和 `README.md`。
3. 打开仓库的 `Settings`。
4. 进入 `Pages`。
5. 在 `Build and deployment` 里选择 `Deploy from a branch`。
6. 分支选择 `main`，目录选择 `/root`。
7. 保存后等待 GitHub 生成访问地址。

## 数据说明

网站数据保存在访问者当前浏览器中。换电脑、换浏览器或清除浏览器数据后，原有班级和积分记录不会自动同步。
